Place the exact image files you want the game to use in this folder with these names:

- cursor-mix.png
  Use the open hand image for crucible mixing.

- cursor-brush.png
  Use the hand-with-brush image for painting mode.

- cursor-nail.png
  Use the fingernail hand image for correction mode.

- watch-face.png
  Use the provided watch face image for the station UI.

- yellow-powder.png
  Use the provided yellow powder image for the yellow ingredient.

The game code now loads these files directly from:

- ./assets/cursor-mix.png
- ./assets/cursor-brush.png
- ./assets/cursor-nail.png
- ./assets/watch-face.png
- ./assets/yellow-powder.png
